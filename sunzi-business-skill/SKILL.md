# SKILL.md - 孙子兵法商业智慧助手

## 描述
基于《孙子兵法》十三篇的深度解读，为商业决策、战略规划、竞争分析提供智慧指导。将2500年前的军事战略智慧转化为现代商业实战工具，帮助企业家和管理者"解密商业密码"。

## 触发条件
当用户询问以下主题时，本技能适用：
- 商业战略与竞争策略
- 企业管理与团队领导
- 谈判技巧与谋略运用
- 市场分析与形势判断
- 危机应对与风险控制
- 创业决策与资源调配
- 孙子兵法的现代应用
- "商战"、"竞争智慧"、"战略思维"、"五事七计"、"知彼知己"等关键词

## 核心智慧体系

### 【计篇】战略规划 - 五事七计框架

> **"兵者，国之大事，死生之地，存亡之道，不可不察也。"**

**商业启示**：战略决策关乎企业存亡，必须全面深入分析后再行动。

#### 五事分析模型

| 兵法原义 | 商业对应 | 分析要点 |
|---------|---------|---------|
| **道** | 企业文化/使命愿景 | 员工是否认同企业目标？团队凝聚力如何？ |
| **天** | 市场趋势/时机 | 外部环境是否有利？行业周期处于什么阶段？ |
| **地** | 行业格局/竞争态势 | 市场位置如何？资源禀赋优劣？ |
| **将** | 领导力/核心团队 | 管理层能力如何？人才储备是否充足？ |
| **法** | 管理制度/执行体系 | 流程是否规范？执行力如何？ |

#### 七计检核清单

1. **主孰有道？** - 哪方领导更有感召力？
2. **将孰有能？** - 哪方管理者更胜任？
3. **天地孰得？** - 哪方更能利用环境优势？
4. **法令孰行？** - 哪方执行力更强？
5. **兵众孰强？** - 哪方资源更雄厚？
6. **士卒孰练？** - 哪方团队更专业？
7. **赏罚孰明？** - 哪方激励更有效？

### 【作战篇】资源与速度 - 兵贵神速

> **"兵闻拙速，未睹巧之久也。"**
> **"故兵贵胜，不贵久。"**

**商业启示**：
- 速度是竞争的核心优势
- 持久战消耗资源，拖得越久风险越大
- 创业和市场竞争讲究"快鱼吃慢鱼"

#### 关键原则

| 兵法原则 | 商业应用 |
|---------|---------|
| **因粮于敌** | 借力打力，利用竞争对手资源，最小化自有投入 |
| **胜敌而益强** | 每次竞争都要让自己变得更强，而非两败俱伤 |
| **兵贵胜，不贵久** | 快速迭代，快速验证，避免陷入消耗战 |
| **拙速胜巧久** | 宁可快速执行粗糙方案，不要拖延追求完美 |

### 【谋攻篇】全胜战略 - 上兵伐谋

> **"不战而屈人之兵，善之善者也。"**
> **"上兵伐谋，其次伐交，其次伐兵，其下攻城。"**

**商业启示**：最高明的胜利是兵不血刃，以最小代价换取最大利益。

#### 战略层级（从优到劣）

| 层级 | 兵法原义 | 商业应用 | 成本收益 |
|-----|---------|---------|---------|
| **上兵伐谋** | 智慧取胜 | 战略定位、品牌塑造、专利壁垒 | 成本最低，收益最大 |
| **其次伐交** | 联盟取胜 | 战略合作、并购整合、生态联盟 | 成本较低，收益较大 |
| **其次伐兵** | 正面对抗 | 价格战、营销战、人才争夺 | 成本较高，收益有限 |
| **其下攻城** | 消耗战 | 恶性竞争、诉讼缠斗、烧钱补贴 | 成本最高，收益最小 |

#### 用兵法则

> **"十则围之，五则攻之，倍则分之，敌则能战之，少则能逃之，不若则能避之。"**

| 兵力对比 | 战术选择 | 商业应用 |
|---------|---------|---------|
| 10:1 | 围而不打 | 绝对优势时，降维打击，不战而胜 |
| 5:1 | 主动进攻 | 显著优势时，正面突破 |
| 2:1 | 分化瓦解 | 优势不大时，各个击破 |
| 1:1 | 灵活应对 | 势均力敌时，避实击虚 |
| 不足 | 保存实力 | 劣势时，战略撤退，保存火种 |

### 【形篇】实力建设 - 先为不可胜

> **"善战者，立于不败之地，而不失敌之败也。"**
> **"不可胜在己，可胜在敌。"**

**商业启示**：
- 先确保自身无懈可击，再等待对手露出破绽
- 能否胜利取决于自己，能否获胜取决于对手是否犯错
- 实力是根基，技巧是枝叶

#### 胜负之道

```
胜兵先胜而后求战
败兵先战而后求胜
```

| 胜者思维 | 败者思维 |
|---------|---------|
| 先有胜算再出手 | 先出手再求胜 |
| 创造必胜条件 | 祈祷好运降临 |
| 控制风险底线 | 孤注一掷 |

### 【势篇】态势营造 - 借势而行

> **"善战者，求之于势，不责于人。"**
> **"故善战人之势，如转木石于千仞之山者，势也。"**

**商业启示**：
- 聪明的领导者借助形势之力，而非单纯依赖个人能力
- 营造"势"，让成功成为自然结果
- 小力量借助大势可以产生巨大影响

#### 造势五法

| 方法 | 兵法原义 | 商业应用 |
|-----|---------|---------|
| **治众如治寡** | 分级管理 | 组织架构清晰，管理幅度合理 |
| **斗众如斗寡** | 统一指挥 | 品牌统一、战略统一、执行统一 |
| **奇正相生** | 常规+创新 | 核心业务+创新业务，稳中求变 |
| **虚实结合** | 示形诱敌 | 营销造势、虚实难辨、信息不对称 |
| **择人而任势** | 选对人、顺势为 | 人岗匹配、借势发力 |

### 【虚实篇】攻守之道 - 避实击虚

> **"避实而击虚。"**
> **"善攻者，敌不知其所守；善守者，敌不知其所攻。"**

**商业启示**：
- 集中优势兵力攻击对手薄弱环节
- 让对手猜不透你的战略意图
- 攻其必救，调虎离山

#### 虚实运用

| 策略 | 说明 | 商业案例 |
|-----|-----|---------|
| **形人而我无形** | 让敌人暴露目标，自己隐藏意图 | 声东击西，佯攻实取 |
| **致人而不致于人** | 掌握主动权 | 主导市场节奏，不被竞争对手牵着走 |
| **攻其必救** | 攻击敌人必救之处 | 切断对手核心利润区 |
| **避锐击惰** | 避开锐气，趁其疲惫 | 新对手入场时观望，待其疲软再反击 |

### 【军争篇】竞争策略 - 以迂为直

> **"军争之难者，以迂为直，以患为利。"**
> **"兵以诈立，以利动，以分合为变者也。"**

**商业启示**：
- 善于将劣势转化为优势，曲折中求直道
- 商业竞争需要谋略，以利益驱动行动
- 灵活分散和集中资源

#### 争利三原则

1. **以迂为直** - 弯道超车，另辟蹊径
2. **以患为利** - 危中寻机，逆境成长
3. **兵以诈立** - 战略保密，出其不意

### 【九变篇】变通之道 - 懂得放弃

> **"途有所不由，军有所不击，城有所不攻，地有所不争，君命有所不受。"**

**商业启示**：
- 知道什么不该做，比知道该做什么更重要
- 战略定力：有所为，有所不为
- 将在外君命有所不受 → 一线管理者要有决策自主权

#### 五不原则

| 原则 | 兵法 | 商业 |
|-----|-----|-----|
| **途有所不由** | 有些路不走 | 有些捷径是陷阱 |
| **军有所不击** | 有些敌人不打 | 有些对手不值得竞争 |
| **城有所不攻** | 有些城池不攻 | 有些市场不进入 |
| **地有所不争** | 有些地盘不争 | 有些份额不值得抢 |
| **君命有所不受** | 君命可违 | 上级指令可据理力争 |

### 【行军篇】团队管理 - 恩威并施

> **"令之以文，齐之以武。"**
> **"卒未亲附而罚之，则不服，不服则难用也。"**

**商业启示**：
- 先建立信任和感情，再用制度约束
- 文治武功结合：文化建设+制度管理
- 奖罚要分明、及时、公正

#### 管理之道

| 要素 | 说明 |
|-----|-----|
| **视卒如婴儿** | 关爱下属，建立忠诚 |
| **视卒如爱子** | 培养人才，成就他人 |
| **厚而不能使** | 不能只宠爱不约束 |
| **爱而不能令** | 不能只有感情没有威信 |
| **乱而不能治** | 不能对混乱视而不见 |

### 【地形篇】环境分析 - 知天知地

> **"夫地形者，兵之助也。"**
> **"知彼知己，胜乃不殆；知天知地，胜乃可全。"**

**商业启示**：
- 深入了解市场环境是决策的基础
- 不同的市场环境需要不同的策略
- 地利可以弥补其他劣势

#### 六种地形

| 地形 | 特点 | 商业应对 |
|-----|-----|---------|
| **通** | 我往彼来皆便 | 先据高阳，占据有利位置 |
| **挂** | 我往不易，彼来易 | 慎重进入，做好退出准备 |
| **支** | 我往彼来皆不便 | 引敌出动，不主动出击 |
| **隘** | 狭窄要道 | 先据要冲，控制关键资源 |
| **险** | 高下险峻 | 占高点，建立壁垒 |
| **远** | 距离遥远 | 避免劳师远征，就地布局 |

### 【九地篇】深入之道 - 背水一战

> **"投之亡地然后存，陷之死地然后生。"**
> **"兵士甚陷则不惧，无所往则固，深入则拘，不得已则斗。"**

**商业启示**：
- 有退路时会犹豫，没退路时会拼命
- 创业团队需要"破釜沉舟"的精神
- 关键时刻要敢于All-in

#### 九地分类

| 地 | 特点 | 应对 | 商业隐喻 |
|---|-----|-----|---------|
| **散地** | 本土作战 | 不宜战 | 舒适区，缺乏斗志 |
| **轻地** | 初入敌境 | 不宜留 | 新市场，快速推进 |
| **争地** | 双方争夺 | 抢占先机 | 核心市场，必争之地 |
| **交地** | 交通要道 | 谨慎通过 | 流量入口，战略要地 |
| **衢地** | 四通八达 | 结交诸侯 | 开放市场，寻找盟友 |
| **重地** | 深入敌境 | 夺取资源 | 深度布局，就地取材 |
| **圮地** | 难行之地 | 快速通过 | 困难市场，不恋战 |
| **围地** | 被围困 | 设谋突围 | 困境突破，创新求生 |
| **死地** | 无路可退 | 拼死一战 | 背水一战，绝地反击 |

### 【火攻篇】特殊战术 - 审慎用计

> **"非利不动，非得不用，非危不战。"**
> **"主不可以怒而兴师，将不可以愠而致战。"**

**商业启示**：
- 没有明确利益就不行动
- 不获全胜就不开战
- 不能因为愤怒而决策，情绪化是商业大忌

#### 火攻五法

| 方法 | 兵法 | 商业应用 |
|-----|-----|---------|
| **火人** | 烧敌人员 | 挖角核心人才 |
| **火积** | 烧敌粮草 | 切断供应链/资金链 |
| **火辎** | 烧敌装备 | 破坏核心资源 |
| **火库** | 烧敌仓库 | 攻击库存/数据 |
| **火队** | 烧敌运输线 | 打击渠道/物流 |

### 【用间篇】情报工作 - 知彼知己

> **"知彼知己者，百战不殆。"**
> **"不知敌之情者，不仁之至也。"**

**商业启示**：
- 信息就是力量，情报决定胜负
- 不重视情报工作是最大的失职
- 商业竞争中信息获取是核心竞争力

#### 五间之法

| 间 | 说明 | 商业应用 |
|---|-----|---------|
| **因间** | 利用敌方乡民 | 市场调研、用户访谈 |
| **内间** | 利用敌方官吏 | 猎头了解对手内部 |
| **反间** | 利用敌方间谍 | 反向获取竞争对手情报 |
| **死间** | 故意泄露假情报 | 战略误导、烟雾弹 |
| **生间** | 派人侦察归来 | 实地考察、试错验证 |
